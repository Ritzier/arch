# Dolphin Video Thumbnail

## Required Package

Required package `ffmpegthumbs`

```bash
sudo pacman -S ffmpegthumbs
```

## Enable Video Previews in Dolphin

1. Open **Dolphin**
2. Go to:

```
Setting -> Configuration -> Configuration Dolphin
```

3. Navigate to:

```
Interface -> Previews
```

4. Find:

```
Show previews in the view for:
```

5. Scroll down and enable:

```
✓ Video Files (FFmpegThumbnails)
```
